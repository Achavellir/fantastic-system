Getting Started

Follow these steps to explore SAMAP's features:

Login with Demo Account

Choose any demo account above and login to access the system. Start with the admin account for full access.

Explore the Dashboard

View real-time security metrics, user statistics, and system health indicators on the main dashboard.

Test API Endpoints

Use the built-in API testing tools to explore user management, security monitoring, and audit log APIs.

View Audit Logs

Check the comprehensive audit trail to see all system activities and user actions being logged.

Generate Reports

Create compliance reports and export data in various formats (PDF, Excel, CSV).

API Endpoints

Key API endpoints available for testing and integration:

POST /api/auth/login User authentication
GET /api/users List all users
GET /api/security/status Security monitoring status
GET /api/audit/logs Audit log entries
POST /api/users Create new user
GET /api/reports/compliance Compliance reports
